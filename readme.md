# manta-ray

A tiny raytracing toolkit for learning purposes.

![example](https://raw.githubusercontent.com/sbuggay/manta-ray/master/demo/example1.png)

## Usage

Building
```
make
```

manta-ray will write a `ppm` to `stdout`. 

Running
```
./build/manta-ray > image.ppm
```

Cleanup
```
make clean
```
